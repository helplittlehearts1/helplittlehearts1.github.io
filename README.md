# 🌟 Kids & Care | Help Little Hearts Website

Welcome to the official repository for the **Help Little Hearts** landing page! 

Our website has a multi-page setup built with plain HTML and CSS, customized with branding textures inspired by `help little hearts.pdf`[cite: 1].

## 📂 Repository File Guide
1. `index.html` - The homepage welcoming visitors.
2. `volunteer.html` - The details for online volunteering tracks.
3. `donate.html` - Information on financial giving options.
4. `contact.html` - Channels to reach out via Instagram or Email.
5. `style.css` - Unified style palette containing the aesthetic custom layouts.
