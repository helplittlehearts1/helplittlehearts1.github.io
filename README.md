# 💖 Help Little Hearts

Welcome to the Help Little Hearts website!

Help Little Hearts is a nonprofit organization dedicated to helping children through kindness, education, community support, and opportunities that help every child thrive.

## 🌸 Features

- Cute pastel pink, blue, and yellow design
- Mobile-friendly layout
- About Us section
- Mission cards
- Volunteer section
- Donation section
- Easy to customize

## 🛠️ Built With

- HTML5
- CSS3


## 🚀 Publishing with GitHub Pages

1. Upload all project files to your GitHub repository.
2. Go to **Settings** → **Pages**.
3. Under **Source**, select the **main** branch.
4. Save your changes.
5. GitHub will publish your website.
