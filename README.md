# 🌟 Kids & Care | Help Little Hearts Website

Welcome to the official repository for the **Help Little Hearts** landing page! 

Our nonprofit is dedicated to uplifting, supporting, and brightening the futures of underprivileged children. This project hosts our cute, pastel-themed website built to share our mission, connect with our community, and recruit passionate online volunteers.

---

## 🎨 Design Theme & Assets
- **Aesthetic:** Cute, cozy, soft pastel illustration style.
- **Palette:** Bubblegum Pink, Sky Blue, and Butter Yellow.
- **Branding:** Based on the official logo design found in `help little hearts.pdf`[cite: 1].

> ⚠️ **Action Required:** Upload your logo file to this repository and make sure it is named exactly what is set in the `index.html` file (e.g., `logo.png`).

---

## 💻 Online Volunteering Program
We make it easy for people all over the world to support our cause digitally. Volunteers can sign up for three distinct tracks:
1. **Virtual Tutoring & Mentorship:** Running educational or storytime events over video calls.
2. **Creative & Design Track:** Crafting digital worksheets and adorable activity packs for distribution.
3. **Awareness Advocates:** Helping spread our message across digital spaces.

## 🚀 How to Launch This Site on GitHub Pages
1. Ensure `index.html`, `style.css`, and your logo image are in the main root directory of this repository.
2. Click on the **Settings** tab at the top of this GitHub repository page.
3. On the left sidebar, click on **Pages**.
4. Under "Build and deployment", set the branch to **main** (or `master`) and folder to **/(root)**.
5. Click **Save**. GitHub will provide a live web link within a few minutes!

## 🚀 Publishing with GitHub Pages

1. Upload all project files to your GitHub repository.
2. Go to **Settings** → **Pages**.
3. Under **Source**, select the **main** branch.
4. Save your changes.
5. GitHub will publish your website.
